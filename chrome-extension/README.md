# 네이버 세션 저장 Chrome 확장 프로그램

네이버 스마트플레이스 로그인 세션을 안전하게 저장하여 AI 답글 생성 기능을 사용할 수 있게 해주는 Chrome 확장 프로그램입니다.

## 📦 설치 방법

### 방법 1: 개발자 모드 설치 (즉시 사용 가능)

1. **Chrome 브라우저 열기**

2. **확장 프로그램 페이지 접속**
   - 주소창에 `chrome://extensions/` 입력 후 Enter
   - 또는 `메뉴(⋮) → 도구 더보기 → 확장 프로그램`

3. **개발자 모드 활성화**
   - 우측 상단의 "개발자 모드" 토글 켜기

4. **확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 버튼 클릭
   - `chrome-extension` 폴더 선택
   - 확인 클릭

5. **설치 완료!**
   - 브라우저 우측 상단에 🔐 아이콘이 표시됩니다
   - 퍼즐 모양(🧩) 아이콘을 클릭하여 확장 프로그램을 고정할 수 있습니다

### 방법 2: Chrome 웹 스토어 (향후 지원 예정)

현재는 개발 중이므로 방법 1을 사용해주세요.

## 🚀 사용 방법

### 1단계: 네이버 로그인

1. [네이버 스마트플레이스](https://new.smartplace.naver.com)에 접속
2. 네이버 계정으로 로그인

### 2단계: 세션 저장

1. 브라우저 우측 상단의 🔐 아이콘 클릭
2. 팝업 창에서 매장 선택
3. "세션 저장하기" 버튼 클릭
4. 완료!

### 3단계: AI 답글 생성 사용

1. 웹사이트의 "AI 답글 생성" 메뉴로 이동
2. 매장 선택 후 리뷰 조회
3. AI 답글 생성 및 포스팅

## 🔧 설정

### API URL 변경 (개발자용)

`popup.js` 파일의 상단에서 API URL을 변경할 수 있습니다:

```javascript
const CONFIG = {
  API_BASE_URL: 'http://localhost:8000', // 개발 환경
  // API_BASE_URL: 'https://your-api.vercel.app', // 프로덕션 환경
  FRONTEND_URL: 'http://localhost:3000'
}
```

프로덕션 배포 시:
1. 주석을 변경하여 프로덕션 URL 사용
2. 확장 프로그램 재로드

## ❓ 문제 해결

### "로그인이 필요합니다" 또는 "401 Unauthorized" 오류

**원인:** 웹사이트에 로그인하지 않았거나 인증 토큰이 저장되지 않음

**해결:**
1. 웹사이트(http://localhost:3000 또는 https://egurado.vercel.app)에 접속
2. Supabase 계정으로 로그인
3. **중요**: 로그인 후 페이지를 새로고침하여 인증 토큰이 localStorage에 저장되도록 하세요
4. 확장 프로그램 다시 열기

### "네이버 로그인 필요" 경고

**원인:** 네이버 스마트플레이스에 로그인하지 않음

**해결:**
1. 네이버 스마트플레이스에 로그인
2. 확장 프로그램 다시 열기

### 확장 프로그램 아이콘이 안 보임

**원인:** 아이콘이 숨겨져 있음

**해결:**
1. 퍼즐 모양(🧩) 아이콘 클릭
2. "네이버 세션 저장" 옆의 📌 아이콘 클릭하여 고정

### "매장 목록을 불러올 수 없습니다" 오류

**원인:** 백엔드 서버가 실행되지 않았거나 인증 문제

**해결:**
1. 백엔드 서버 실행 확인
2. API URL이 올바른지 확인 (popup.js의 CONFIG.API_BASE_URL)
3. 웹사이트에 로그인 후 페이지 새로고침
4. 확장 프로그램 재로드 (chrome://extensions/ → 새로고침 버튼)

## 🔒 보안

- 모든 쿠키 데이터는 HTTPS를 통해 암호화되어 전송됩니다
- 서버에는 암호화된 형태로 저장됩니다
- 확장 프로그램은 네이버 도메인의 쿠키만 접근합니다
- 사용자의 비밀번호는 절대 저장하지 않습니다
